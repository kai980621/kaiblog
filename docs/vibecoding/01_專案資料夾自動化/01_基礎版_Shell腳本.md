## 💡 解決什麼問題？
每次開啟新專案時，都要手動按右鍵新建一堆重複的資料夾（如素材、專案檔、輸出），既浪費時間又容易不統一。這個工具可以讓你「一鍵」完成所有準備工作！

---

## 🤖 AI 提示詞 (Prompt) Template
複製以下提示詞，根據你的需求修改後貼給 AI

```text
這是我目前的專案工作流程，請幫我寫一個自動建立資料夾的腳本。
請給我電腦原生可以執行的版本（Mac 用 Shell Script, Windows 用 Batch 檔）。

### 我的需求：
1. 腳本啟動時，詢問我「專案編號」與「專案名稱」。
2. 根據輸入，建立一個主資料夾。
3. 在主資料夾內，自動建立以下子目錄結構：
    - `00_Admin` (放報價單、合約)
    - `01_Materials` (放原始素材)
        - `Footage`
        - `Audio`
    - `02_ProjectFiles` (放 Pr/Ae/Figma 專案檔)
    - `03_Exports` (輸出檔案)
4. 在主資料夾內建立一個 `info.txt`，並在裡面自動寫入剛剛輸入的專案名稱與建立日期。
5. 建立完成後，自動用檔案總管/finder開啟該資料夾。

### 我要的結果：
1. 幫我生成腳本
2. 簡單介紹運行的原理
3. 告訴我怎麼執行
```
