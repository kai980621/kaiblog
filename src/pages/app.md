# 📱 KAI 電腦裡有什麼 `/app`

- 最後更新：2026-05-09
- **總共 `37` 個 軟體**
> 這是我在 Mac 上的軟體清單，<a>藍色字為最近更新的軟體</a>

### 01. Affinity : 製作縮圖常用，[可以看這篇文章](/blog/affinity)   
### 02. Anytype : 超好用開源筆記軟體，載入速度極快   
### 03. Aria Player : Finale 的專屬音樂播放器  (可以刪掉嗎?)
### 04. Audacity : 免費開源超好用的音訊編輯器    
### 05. AutoSubs : 快速生成字幕
### 06. Blip : 開源跨平台傳檔，遠距離也沒問題   
### 07. Brave Browser : 具有隱私保護的開源瀏覽器   
### 08. Cubase 15 : 主要的編曲軟體，教學資源不錯   
### 09. DaVinci Resolve : 找不到比他更好的調色與剪輯軟體了   
### 10. Discord : 聊天與語音頻道首選   
### 11. Finale 27 : 停止提供更新了，但我依然習慣它   
### 12. Github Desktop : 管理本站 GitHub 倉庫的好幫手   
### 13. Groove Agent SE : Cubase 內建的鼓機   
### 14. HALion Sonic : Cubase 內建音源，耗電量低   
### 15. IINA : 開源萬能播放器，Mac 必裝   
### 16. Keynote : 蘋果簡報工具   
### 17. Kontakt 8 : 強大的採樣音源主機   
### 18. Line : 必要但笨重的通訊軟體   
### <a>19. LM Studio : 運行本地AI模型的工具</a>
### 20. Logic Pro : 蘋果自家 DAW，想刪除了！   
### 21. Microsoft Excel : 強大的試算表工具   
### 22. Microsoft PowerPoint : 簡報必備工具   
### 23. Microsoft Word : 業界標準文書軟體   
### 24. Musescore 4 : 開源打譜軟體   
### 25. NDM : 下載神器，速度很快   
### 26. Numbers : 蘋果表格工具   
### 27. OBS : 強大的開源螢幕錄影與直播軟體   
### <a>28. Obsidian : 強大的開源筆記 (MarkDown) 編輯軟體</a>
