# 📱 KAI 電腦裡有什麼 `/app`

- 最後更新：2026-03-27
- **總共 `42` 個 軟體**
> 這是我在 Mac 上的軟體清單。我最近清除了一些...不然太多了



## 🎵 DAW 與音訊編曲
1. **Ableton Live** ㄧ 業界標準的 Live 演出與編曲軟體
2. **Cubase 15** ㄧ 主要的編曲軟體，教學資源不錯
3. **Logic Pro** ㄧ 蘋果自家 DAW，內建音色非常紮實
4. **Audacity** ㄧ 免費開源超好用的音訊編輯器
5. **UVR 5** ㄧ 去人聲神器，使用本地模型

## 🎹 虛擬音源與採樣器
6. **Kontakt 8** ㄧ 強大的採樣音源主機
7. **Aria Player** ㄧ Finale 的專屬音樂播放器
8. **Groove Agent SE** ㄧ Cubase 內建的鼓機
9. **HALion Sonic** ㄧ Cubase 內建音源，耗電量低
10. **Vital** ㄧ 視覺化很強的譜面合成器

## 🎼 樂譜製作
11. **Dorico 6** ㄧ 現代化且排版極其精美的專業製譜軟體，**最近因為操作困難想放棄了．．．**
12. **Finale 27** ㄧ 停止提供更新了，但我依然習慣它
13. **Musescore 4** ㄧ 開源打譜軟體

## 🎬 影片剪輯與處理
14. **Davinci Resolve** ㄧ 找不到比他更好的調色與剪輯軟體了
15. **Final Cut Pro** ㄧ 蘋果原生剪輯軟體，剪接流暢度極高
16. **OBS** ㄧ 強大的開源螢幕錄影與直播軟體
17. **YTDownloader** ㄧ 下載 Youtube 影片很方便
18. **Affinity** ㄧ 製作縮圖常用，[可以看這篇文章](/blog/affinity)

## 📝 開發與程式設計
19. **Antigravity** ㄧ Google 官方的 AI 程式碼編輯器
20. **Github Desktop** ㄧ 管理本站 GitHub 倉庫的好幫手

## 📝 文書與筆記
21. **Anytype** ㄧ 超好用開源筆記軟體，載入速度極快
22. **Microsoft Word** ㄧ 業界標準文書軟體
23. **Microsoft PowerPoint** ㄧ 簡報必備工具
24. **Microsoft Excel** ㄧ 強大的試算表工具
25. **Keynote** ㄧ 蘋果簡報工具
26. **Numbers** ㄧ 蘋果表格工具
27. **Pages** ㄧ 蘋果排版工具

## 🛠️ 系統維護與優化
28. **Pearcleaner** ㄧ 大推！乾淨卸載 App 的利器
29. **Raycast** ㄧ 取代 Spotlight 的生產力核心
30. **IINA** ㄧ 開源萬能播放器，Mac 必裝

## 📂 網路、檔案與虛擬化
31. **Brave Browser** ㄧ 具有隱私保護的開源瀏覽器
32. **CrossOver** ㄧ 在 Mac 上執行 Windows 程式
33. **MacroMachine** ㄧ 達芬奇上製作模板的好工具
34. **NDM** ㄧ 下載神器，速度很快
35. **Parallels Desktop** ㄧ Windows 虛擬機，效能強大

## 💬 社群通訊
36. **Discord** ㄧ 聊天與語音頻道首選
37. **Line** ㄧ 必要但笨重的通訊軟體
38. **Blip** ㄧ 開源跨平台傳檔，遠距離也沒問題

## 🎮 遊戲與創作
39. **Roblox** ㄧ 我唯一玩的遊戲
40. **Roblox Studio** ㄧ 做遊戲非常簡單，推薦克拉克

## 💩 官方資源管理
41. **Steinberg Download Manager** ㄧ Steinberg 產品資源下載中心
