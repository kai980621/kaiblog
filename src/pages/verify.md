# 🛡️ 伺服器認證程序

歡迎來到 **KAI STUDIO** 社群。為了維護討論品質並防範惡意帳號，請你在加入前完成以下認證手續。


## 🚀 驗證流程 (Verification)

1.  **詳閱規則**：請先完整閱讀下方守則。
2.  **訊息確認**：回到 Discord 的 **#入群認證** 頻道，在規則訊息下按下 **✅**。
3.  **填寫表單**：點擊下方連結前往 OpenForms 填寫基本資料。
4.  **完成認證**：填寫完後，回到本頁面點擊最下方的「提交認證」按鈕。


## 📜 伺服器守則 (Server Rules)

### 1. 尊重與禮節 (Respect & Etiquette)
- **友善互動**：嚴禁任何形式的歧視、霸凌、人身攻擊或惡意挑釁。
- **尊重隱私**：未經他人同意，禁止散佈私人對話內容、照片或其他個人資訊 (Doxxing)。
- **溝通氛圍**：理性討論，避免過度的政治、宗教或敏感話題爭議。

### 2. 內容規範 (Content Standards)
- **嚴禁非法內容**：禁止傳送任何違反法律、惡意程式、釣魚連結或破解盜版軟體。
- **分級控制**：禁止在非特定頻道（如 NSFW 區）發送色情、暴力或令人不適的影像。
- **版權尊重**：分享作品、代碼或教學時，請盡量註明出處，尊重原創者。

### 3. 頻道秩序 (Chat Discipline)
- **禁止刷屏**：請勿連續傳送重複的訊息、貼圖、大量 Emoji 或無意義的噪音。
- **頻道專用**：請根據各頻道的用途發言（例如：音樂製作討論、技術除錯）。
- **廣告宣傳**：禁止未經許可的私訊推銷或張貼其他伺服器邀請連結。

### 4. 帳號與身分 (Accounts)
- **單一帳號**：原則上每人僅限使用一個帳號加入，嚴禁使用分身帳號規避懲處。
- **名稱規範**：請勿使用包含辱罵、不雅文字或偽裝成管理員的暱稱。

### ⚠️ 違規處理機制
- **初犯/輕微**：口頭警告或暫時禁言 (Mute)。
- **再犯/嚴重**：強制踢出 (Kick) 或永久封鎖 (Ban)。
